# Steam Count

Steam Count is a Python program that displays the Steam game you are currently playing, along with the number of players currently in the game. It utilizes the Steam Web API to fetch the necessary data. Steam Count will fetch the game you are currently playing and display its name and the number of players currently in the game. The program will update this information every time you exit or launch a new game.

## Prerequisites

Before using Steam Count, you need to have the following:

- A Steam API key, which you can obtain at [Steam Community](https://steamcommunity.com/dev/apikey).
- Your Steam ID, which can be found in your Steam account details.

## Installation
- Download the zip file from the most recent release
- Extract the zip file and run Steam Count.exe
